## 01. 힙이란?  

### 힙
두 개의 조건을 만족하는 트리  
1. 형태 속성: 힙은 완전 이진 트리이다(높이: $O(lg(n))$  
2. 힙 속성: 모든 노드의 데이터는 자식 노드들의 데이터보다 크거나 같다  

힙을 활용하는 두 가지 방법  
1. 정렬 문제 해결  
2. 우선순위 큐 구현

<br/><br/>

## 02. 정렬 문제  

### 정렬
- 여러 개의 데이터 요소들을 특정 순서로 배치하는 것  
- 예시)
```python
[4, 1, 6, 2, 8, 5]

[1, 2, 4, 5, 6, 8]  # 오름차순 정렬

[8, 6, 5, 4, 2, 1]  # 내림차순 정렬
```

### 정렬 알고리즘: 데이터를 재배치하는 구체적인 방법  
- 삽입 정렬
- 선택 정렬
- 퀵 정렬
- 합병 정렬
등
