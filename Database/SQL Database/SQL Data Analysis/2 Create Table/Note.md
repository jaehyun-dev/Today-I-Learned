## 01. CSV 파일로 테이블 생성하기

데이터베이스 안에 테이블을 생성하는 2가지 방법
- SQL문으로 생성하는 방법
- CSV파일을 임포트(import)해서 테이블로 만드는 방법

### CSV 파일
- Comma Separated Values의 약자
- 모든 값들이 콤마(,)로 구분된 형식의 내용을 가진 파일
- 보통 가장 첫 번째 줄에 컬럼 이름들이 적혀있음

### CSV 파일을 테이블로 만들기
- 데이터베이스 이름 오른쪽 클릭, Table Data Import Wizard
